Smart Banking System (Java OOP Project)

Description

The Smart Banking System is a console-based Java application designed to simulate basic banking operations. It demonstrates core Object-Oriented Programming (OOP) principles along with file handling to ensure data persistence.

This project follows a modular and structured design approach, making it suitable for beginners to strengthen their programming fundamentals and present a professional project on GitHub.

---

Features

User Management
Account registration
Secure login system

Banking Operations
Deposit money
Withdraw money with validation
Balance inquiry

Transactions
Transaction history tracking
File-based transaction storage

Data Persistence
User data stored in files
Transactions saved permanently

---

OOP Concepts Used

Encapsulation
Abstraction
Modular class design
Inheritance and polymorphism (optional for extension)

How to Run

Clone the repository
git clone https://github.com/SyedAbdullahQureshi/SmartBankingSystem-Java.git

Open the project in any Java IDE such as IntelliJ IDEA, Eclipse, or Visual Studio Code

Compile the program
javac Main.java

Run the application
java Main
